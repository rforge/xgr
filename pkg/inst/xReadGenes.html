xReadGenesR Documentation

Function to prepare a gene_info RData file

Description

xReadGenes is supposed to prepare a gene_info RData file.

Usage

xReadGenes(gene_info.file = NULL, output.prefix = NULL, output.dir =
"./")

Arguments

gene_info.file

an input file for gene_info

output.prefix

a prefix for the output file

output.dir

the output directory

Value

an EG object

Note

None

See Also

xReadGenes

Examples

## Not run: 
# Load the library
library(XGR)

EG <- xReadGenes(gene_info.file="GeneInfo.9606",
output.prefix="org.Hs.eg")

## End(Not run)
